# About Project

Fitnesshub is a Saas product.

Fitnesshub aims to deliver a fitness club management software for gyms and clubs. It is a modern SaaS platform for seamless gym management. It helps gym owners and trainers streamline daily operations, track client fee 7 workouts etc.., and enhance member experience.

Gym owners can manage their organization, Gym members can see their membership information.

This is a prototype version in nextjs which will be planned to migrate to nextjs + go api

## Techstack

- Nextjs - Framework
- Shadcn UI - UI framework
- TailwindCSS - UI Framework
- BetterAuth - Authentication and user Management
- DrizzleORM - Database operations
- Postgres - Database
